# Market Bot - Automated Telegram Updates

## 🚀 Features
- **XAUUSD & BTCUSD Priority** - Get the most important market data first
- **Free Market Data** - Uses free APIs and RSS feeds from trusted sources (Bloomberg, Reuters, ForexLive)
- **Automated Telegram Updates** - Sends daily updates at 8 AM and 4 PM UTC
- **Price Alerts** - Get notified when prices move beyond configured thresholds
- **Trading Signals** - Basic technical analysis signals
- **Web Dashboard** - Monitor everything in real-time

## 📦 Setup Instructions

### 1. Create Telegram Bot
1. Message @BotFather on Telegram
2. Send `/newbot` and follow instructions
3. Save your bot token
4. Add bot to your group
5. Get group chat ID (use @getidsbot)
